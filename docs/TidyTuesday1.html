Inequality across countries

Graphing the Gini coefficients of 52 countries over time

Author

Abigail Krenz

Published

December 6, 2025

Using a World Bank data set, I visualized the inequality, as measured by the Gini coefficient, of 52 countries across the last few decades. Historically, South Africa has experienced the highest levels of inequality, followed by Brazil, although inequality in both countries has decreased over time. Countries with low levels of inequality included Iceland, Finland, and Switzerland.

Show the code 
library(tidyverse)
library(plotly)

#Reads data
tuesdata <- tidytuesdayR::tt_load('2025-08-05')
income_inequality_processed <- tuesdata$income_inequality_processed
income_inequality_raw <- tuesdata$income_inequality_raw

plot <-ggplot(data=income_inequality_processed, 
              aes(x=Year, 
                  y=gini_mi_eq, 
                  color=Entity, 
                  text = paste('Country:', Entity))) +
  geom_line() + 
  labs(title="Inequality in countries across time", x="Year", y="Gini Coefficient", color="Country", caption = "Hello") + 
  theme_minimal()

#Uses ggplotly to add interactive hover labeling
ggplotly(plot, tooltip = "text")

Country with the highest inequality by far was South Africa. Those with low inequality included Iceland, Finland, and Switzerland.
